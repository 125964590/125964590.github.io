# 网络配置

这里直接将网卡固定,稍后会配置成桥接的网卡

网卡配置位置

> /etc/netplan/50-cloud-init.yaml

```yaml
network:
    ethernets:
        enp5s0f1:
            addresses:
            - 10.1.12.44/23
            dhcp4: 'no'
            gateway4: 10.1.13.254
            nameservers:
                addresses:
                - 114.114.114.114
   version: 2
```
