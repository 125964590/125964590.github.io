- Mapping:对请求进行匹配
- Handler:队请求进行加工
- Filter:使用责任链队请求进行切面处理
- 返回结果

## Route
是Gateway中的基础组件之一,==Route==是gateway中的一种抽象包含如下几个部分:
1. id:标识符区别于其他的Route
2. uri:路有志向的目的地
3. order:优先级的一个定位
4. predicate:匹配路由的一个前置条件
5. gateway filters:过滤器切面逻辑
