# 操作流程

## test

```shell
kubectl create secret tls https-test --cert=3069811_gateway-paas-test.facethink.com.pem --key=3069811_gateway-paas-test.facethink.com.key -n godhand-paas-test


kubectl get ingresses -n godhand-paas-test  -o yaml godhand-paas-test-gateway --export > ingress.yaml

  tls:
  - hosts:
    - gateway-test.facethink.com
    secretName: https-dev


   annotations:
    nginx.ingress.kubernetes.io/ssl-redirect: "false"
```
